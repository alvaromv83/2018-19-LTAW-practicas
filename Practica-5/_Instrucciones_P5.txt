# Instrucciones Práctica 5
==========================

Requisitos previos:
------------------
    1) Instalar Node.js:
          & sudo apt-get install curl
          curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
          & sudo apt-get install nodejs

    2) Instalar Electron (framework en el que se basa esta práctica):
          & npm install electron

    3) Instalar socket.io-client (API que se usa para la comunicación
       bidireccional entre cliente y servidor a través de websocket):
          & npm install socket.io

Probar chat:
------------
    1) Lanzar servidor:
          $ node chat-server.js
    2) Abrir app de cliente:
          $ npm start
