Enunciado Práctica 1
--------------------

Construir un servidor web usando Node.js, que sirva las páginas de la web de una
tienda. Las páginas serán estáticas (en HTML), y consistirán en textos, fotos y
vídeos enlazados, compartiendo una hoja de estilo

Componentes obligatorios:

    * Usar el móudlo http de Node.js
    * Documentos HTML, horas de estilo (.css), vídeo, audio e imágenes
